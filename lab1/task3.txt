Параметр charset нужен для того, чтобы определить кодировку страницы.

Если сharset не будет задан, или будет задан неверно, 
то по умолчанию браузер должен принять charset ISO-8859-1.
Или же, например, IE попытается определить то кодирование,
которое использует пользователь.

На данной html странице произошло несоответствие кодировок.
Это решается добавлением тега meta c атрибутом charset="utf-8" в самое начало html файла.
